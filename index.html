Ejercicio

Este es el contenedor principal, el objetivo es que todo se vea bien aun cambiando las resoluciones, deberas cambiar los colores y tamaños explotando las variables y funciones de SASS

Deberás obtener la lista de asistencia usando la API ubicada en http://laboratoria.cuadra.co:9339/api/v1/students/ y agregarle estilos con SASS

Asistencia:

    Deberás

    Crear una función para calcular el width % con base en que col es, ej col-1 = 8.33% ,

    También

    Implementar los colores de laboratoria usando la función rgb/a

    Por último

    Implementar anidación de máximo 4 niveles hacia abajo para facilitar la lectura del archivo
